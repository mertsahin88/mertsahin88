### About me

I build the developer tools that put AI in front of enterprise code. Focused on Large Language Models and code generation for SAP/ABAP systems.

My path is unusual: 15+ years as an SAP Technical Architect (Deloitte, AkzoNobel, Yasar Holding) shipping S/4HANA Cloud greenfield implementations, RAP/CAP applications, and BTP integrations — then a deliberate transition into AI/ML starting in 2023.

### Featured projects

#### [abapmcpserver](https://github.com/mertsahin88/abapmcpserver)
Model Context Protocol (MCP) server connecting AI assistants (Claude Desktop, VS Code, ...) to SAP ABAP systems via the ADT REST API. **69 tools** for the full ABAP development lifecycle — read, write, refactor, RAP/CDS generation, debugger, ATC, unit tests. Built with TypeScript, MCP SDK, SAML/Kerberos SSO, undici, vitest.

#### [sapien](https://github.com/mertsahin88/sapien)
Self-learning SAP cognitive brain: HTTP API + MCP adapter backed by a Neo4j knowledge graph. Auto-discovers ABAP repositories via ADT, parses with @abaplint/core AST, infers an ontology of **242 node types** and **54+ relations**, runs clean-core compliance with typed evidence, serves hybrid retrieval (graph + semantic + LLM) to dev tools and AI assistants. **980+ tests** across 64 files. TypeScript, Neo4j, Node.js 22.
